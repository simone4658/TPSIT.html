HTML

HTML, che significa letteralmente "Hyper Text Markup Language" ("Linguaggio a marcatori per ipertesti"), è un linguaggio che permette di impaginare e formattare pagine collegate fra di loro, ovvero creare dei siti web.

Javascript

JavaScript è un linguaggio di scripting lato client utilizzato per rendere interattive le pagine web. JavaScript, insieme ad HTML e CSS, è una delle tecnologie principali della programmazione web front end. Infatti JavaScript gestisce il comportamento degli elementi dinamici di un sito web, ovvero come il contenuto (HTML) e lo stile grafico (CSS), che reagiscono e si modificano sulla pagina web.

CSS

CSS, o Cascading Style Sheets, è un linguaggio usato per programmare la resa grafica di documenti scritti in un linguaggio di markup, come per esempio HTML. Le classi CSS sono usate per specificare attributi grafici come font, dimensione, colore, spaziatura, bordo e posizione degli elementi all'interno di una rappresentazione web.

Sistemi

Un sistema può essere definito come l'unità fisica e funzionale, costituita da più parti o sottosistemi (tessuti, organi o elementi ecc.) interagenti (o in relazione funzionale) tra loro (e con altri sistemi), che formano un tutt'uno in cui ogni parte dà il proprio contributo per una finalità comune (o un obiettivo identificativo).
